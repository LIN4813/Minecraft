核心 特性

模组整合

精选优质模组组合，无冲突、高兼容，大幅拓展游戏内容与玩法，适配最新版本。

材质优化

定制化像素材质包，保留原版风格的同时提升视觉质感，低配置也能流畅运行。

服务端支持

内置轻量化服务端配置，一键搭建私人服务器，支持多人联机游玩。

安全稳定

去除恶意代码，优化内存占用，降低崩溃概率，保障游戏运行稳定性。

自定义配置

可视化配置界面，无需修改代码即可调整游戏参数、模组开关、按键绑定等。

一键安装

简化安装流程，自动检测游戏路径，一键部署全部资源，新手也能快速上手。

快速 安装

  1. 1

    克隆仓库

    打开终端/命令提示符，执行以下命令克隆 GitHub 仓库：

    git clone https://github.com/LIN4813/Minecraft.git
  2. 2

    检查环境

    确保已安装 Java 8+（适配 Minecraft 运行），并配置好环境变量。

  3. 3

    运行安装脚本

    进入仓库目录，执行安装脚本（Windows 双击 install.bat，Linux/Mac 执行 ./install.sh）：

    cd Minecraft && ./install.sh
  4. 4

    启动游戏

    安装完成后，运行 start.bat（Windows）或 ./start.sh（Linux/Mac）启动游戏。
